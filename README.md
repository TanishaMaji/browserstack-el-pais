# El País Selenium Testing with BrowserStack
This project scrapes the first 5 articles from El País Opinion section,
prints Spanish titles & content, downloads cover images, translates titles
to English using Rapid Translate API, and analyzes repeated words.
Rapid Translate API has a limited of 50 request per day. 

## How to Run Locally
Python main.py
Rapid API credentials are required.

## How to Run on BrowserStack Parallelly (Cross-Browser Testing)
python browserstack_parallel.py
BrowserStack credentials are required.
